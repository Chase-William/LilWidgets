relativeShadowSigma
The shadow sigma used for the drop shadow which is relative to the strokeRatio. This helps the shadow resize correctly. Without this the shadow can be come obnoxious when scaling from a larger size to a smaller one.
relativeStrokeWidth
The last determined value of the product of StrokeWidth * strokeRatio. This is used for resizing. Updates come from the canvas_PaintSurface(Object, SKPaintSurfaceEventArgs) method.
shadowColor
The color used for the drop shadow.
ShadowColorProperty
BindableProperty for the ShadowColor property.
stopwatch
Used in conjunction with the StartTimer(TimeSpan, FuncBoolean)'s lambda inside the PercentValuePropertyChanged(BindableObject, Object, Object) to determine the amount of time passed from the last time being restarted. This is used as deltaTime to help smooth out animations and pull away from using frame-rate as a method of determining how much a target value should change over a given amount of time.
strokeRatio
The ratio of the canvas size and StrokeWidth to be used when resizing. This value only updates when the StrokeWidth is changed.

BindableProperty for the StrokeWidth property.

textWidth

txtSpaceRatio
The ratio of the canvas size and the TextMargin to be used when resizing. This value only updates when the TextMargin is changed.
