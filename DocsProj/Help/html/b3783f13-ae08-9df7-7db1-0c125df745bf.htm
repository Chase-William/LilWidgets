ProgressWidget Class

The Progress Widget is designed to show a percentage of a goal or objective. This can be anything from a quiz score where the user attained a 90%, or even a loading bar where the loading process’s progress can be given.

Is the time required to go from a PercentProgressValue of 0.0 to a PercentProgressValue of 1.0. Therefore if PercentProgressValue is changing from 0.0 to 0.5 and the Duration is set to 2000 milliseconds; it will take one second to complete the animation. The equation looks like this (relativeDuration == milliseconds * difference).
